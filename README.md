# HR-Analytics-Dashboard
📊 HR Analytics Dashboard – Power BI

🔍 About the Project
This project analyzes employee attrition data using Power BI.
The goal is to help HR teams understand why employees leave and how to improve retention.

🛠️ Tools Used
Power BI – Dashboard & Visualizations
Excel/CSV – Dataset

📌 Key Insights
📉 Attrition rate is 16.1% (237 out of 1,470 employees)
🏢 R&D and Sales departments have the highest attrition
⏱️ Most employees leave within the first 1–3 years
👶 18–25 age group has the highest attrition count
🎓 Life Sciences graduates make up the largest share of attrition
